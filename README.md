# tic-tac-toe-frontend

## Overview
This is a simple Tic Tac Toe game implemented in C++. It allows two players to take turns marking spaces in a 3Ã—3 grid until a winner is determined or the game ends in a draw.

## Features
- Two-player mode
- Displays the game board after each move
- Detects winning conditions and draws
- Simple command-line interface

## Technologies Used
- Programming Language: C++

## Usage
- Players take turns entering their moves.
- The game board updates after each move.
- The game announces the winner or a draw.
